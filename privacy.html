Privacy Policy

Digital Transit Pass is an internal application developed by Prasarana Malaysia Berhad, intended solely for use by authorized personnel.

📷 Camera Usage

This app accesses the device's camera for the purpose of scanning identification documents (e.g., MyKad) to extract text using Optical Character Recognition (OCR).

  • Captured images are securely transmitted to a third-party OCR service for text recognition.
  • The images are not stored on the device after processing.
  • The OCR service is used strictly for functional purposes within the app's workflow.

🔐 Data Collection & Handling

  • No personal data is stored or shared beyond the OCR process.
  • All data transmission is done over secure, encrypted channels (HTTPS).
  • The third-party service is contractually obligated to handle data in compliance with applicable privacy regulations.

👥 Scope & Usage

This application is distributed for internal use only and is not intended for public use or distribution.
